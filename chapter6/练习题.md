### 1

```c
i = 1;
while (i <= 128) {
    printf("%d ", i);
    i *= 2;
}
// >1 2 4 8 16 32 64 128 %
```

### 2

```c
i = 9384;
do {
    printf("%d ", i);
    i /= 10;
} while (i > 0);
// >9384 938 93 9 %
```

### 10

在循环体的末尾增加一条空语句，并为其添加一个标号，使用`goto 标号`替换 continue 语句。

### 11

```c
sum = 0;
for (i = 0; i < 10; i++) {
    if (i % 2)
        continue;
    sum += i;
}
printf("%d\n", sum);
// >20
```
